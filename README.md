# Errors

Errors package. 
* Convert simple errors to HTTP/GRPC codes 
* Reach errors with values using `WithPayloadKV` 
* Add values to log with `WithLogKV`
* Support tags
* Support callbacks

### Types
* Internal error
* Entity not found
* Entity already exists
* Invalid request
* Unprocessable entity
* Request timeout
* Authentication required
* Access denied
